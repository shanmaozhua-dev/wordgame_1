# 03_动画接入规范

本文件规定动画、序列帧、音效和视觉效果如何接入。

## 动画来源

- 所有动画只允许来自 `D:\文字游戏`。
- 接入时在 `level_manifest.json` 中记录原始路径。
- 复刻工程内可以使用 `res://Scenes/Animations/...`、`res://Sprites/...`、`res://Sounds/...`，但必须能追溯到 `D:\文字游戏`。
- 不允许临时自创动画。缺失资源必须记录为风险。

## 两类动画接入方式

### 1. 已封装 `.tscn` 动画

常见目录：

```text
res://Scenes/Animations/
```

例如：

```text
res://Scenes/Animations/Backspace.tscn
res://Scenes/Animations/Pull.tscn
res://Scenes/Animations/Push.tscn
res://Scenes/Animations/SentenceLegal_new.tscn
res://Scenes/Animations/split_animation.tscn
res://Scenes/Animations/merge_new_event_animation.tscn
```

规则：

- 优先复用源项目已有 `.tscn`。
- 不要把动画节点挂到随机位置；确认源代码中原本挂在哪个节点下。
- 拆字/合字/删除/推拉动画已有源逻辑时，复刻应优先对齐源逻辑。

### 2. `play_sprite_animation` 序列帧

源项目事件常用命令：

```text
@[play_sprite_animation] {
  "frame_count": 30,
  "path": "res://Sprites/door/door_open.png",
  "target": "進來門",
  "time_sec": 1
}
```

可选字段：

| 字段 | 说明 |
|---|---|
| `frame_count` | 总帧数 |
| `path` | 序列帧贴图 |
| `target` | 目标事件名；省略时通常是当前事件 |
| `time_sec` | 播放时长 |
| `reverse` | 是否倒放 |
| `width` | 每行帧数，默认常见为 10 |
| `wait` | 是否等待播放完成 |

## 位置基准

- 普通事件本体 `position` 是格子左上角。
- `WordSprite` 默认 `position = Vector2(30, 30)`。
- 动画如果挂到 `WordSprite`，坐标基准通常是文字中心。
- 动画如果挂到事件节点，坐标基准通常是事件左上角。
- 句子合法动画 `play_sentence_legal_animation(pos, text_count, is_h, ...)` 使用像素坐标，源码中通常由 `bound.position * 60` 加成。

## 音效接入

常用命令：

```text
@[play_se] {"path":"res://Sounds/se/第一章 音效/SE_1_3_door_open.wav"}
```

规则：

- 路径必须来自源项目。
- 同一动画若源项目同时播放音效，必须一起记录。
- 位置音效使用源项目已有命令或脚本，不要自行替代。

## BGM 和环境音

地图级字段：

```text
has_bgm = true
bgm_path = "res://Sounds/bgm/ch1/BGM_1_1_room1.ogg"
bgm_transition_type = "fade"
has_env = true
env_path = "res://Sounds/env/..."
```

规则：

- 每张地图在 `level_manifest.json` 记录 BGM 和环境音。
- 没有 BGM 的地图也要显式写 `has_bgm: false`。
- 不确定是否继承上一张地图时，在交接风险中写明。

## 触发条件记录

每个动画必须说明触发条件：

- 自动触发：哪个 `AUTO` 事件。
- 玩家触发：哪个门/触发器/NPC。
- 规则触发：哪个句子、拆字、合字。
- 删除触发：哪个 `backspace_command` 或 `can_delete` 事件。
- 失败触发：例如 Backspace 失败、拆字失败。

## 禁止事项

- 禁止因为源动画复杂而自创简化动画。
- 禁止把动画贴图裁剪后不记录。
- 禁止只接视觉不接源音效。
- 禁止只写“用了门动画”，必须写具体路径。
- 禁止把动画节点放到会随布局偏移的 UI 节点里，除非源项目就是这样。

## Manifest 记录示例

```json
{
  "required_resources": {
    "animations": [
      {
        "source": "D:/文字游戏/Scenes/Animations/door_open.tscn",
        "target": "res://Scenes/Animations/door_open.tscn",
        "trigger": "Event: 門 / command: play_sprite_animation",
        "position_anchor": "event_word_center"
      }
    ],
    "sprites": [
      {
        "source": "D:/文字游戏/Sprites/door/door_open.png",
        "target": "res://Sprites/door/door_open.png"
      }
    ],
    "audio": [
      {
        "source": "D:/文字游戏/Sounds/se/第一章 音效/SE_1_3_door_open.wav",
        "target": "res://Sounds/se/第一章 音效/SE_1_3_door_open.wav"
      }
    ]
  }
}
```
