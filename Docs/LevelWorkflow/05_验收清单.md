# 05_验收清单

每张地图交付前必须逐项检查。没有通过的项写进 `handoff.md`，不能只口头说明。

## 基础文件

- [ ] 地图场景存在于 `Scenes/Maps/<章节>/<地图名>.tscn`。
- [ ] `level_manifest.json` 已填写。
- [ ] `handoff.md` 已填写。
- [ ] 使用的源地图路径已记录。
- [ ] 使用的动画、贴图、音效源路径已记录。

## 地图结构

- [ ] 根节点结构与源项目一致或有明确说明。
- [ ] 有 `MainMap`，且实例/脚本正确。
- [ ] `MainMap.bound` 设置正确。
- [ ] `TileSet` 正确。
- [ ] 可走区域有 tile。
- [ ] 不可走区域不会误通行。
- [ ] `TopLayer/FixedMap` 没有缺失。

## 坐标与文字

- [ ] 所有普通事件 `position = now_pos * 60`。
- [ ] 文字视觉中心在格子中心。
- [ ] 单字没有跑出 `60x60` 格子。
- [ ] 多字/竖排文本按 60 像素步长对齐。
- [ ] `WordSprite.position` 没有被误改成 `(0,0)`。
- [ ] 大字事件或特殊文本有单独说明。

## 玩家与流程

- [ ] 玩家出生点正确。
- [ ] 从上一张地图能进入本地图。
- [ ] 本地图主流程能走完。
- [ ] 能切到下一张地图。
- [ ] 死亡、重来、返回标题等系统流程没有被破坏。

## 事件

- [ ] 门事件存在。
- [ ] 触发器位置正确。
- [ ] `event_trigger_action` 正确。
- [ ] 事件 `layer` 正确，阻挡符合源项目。
- [ ] `can_push`、`can_delete`、`can_split` 对齐源项目。
- [ ] `exist_condition` 对齐源项目。
- [ ] `need_be_saved` 对齐源项目。
- [ ] 命令中的 `target` 都能找到对应事件。
- [ ] `copy_from_event` 指向有效事件。

## 剧情命令

- [ ] `commands` 能被解析。
- [ ] `@[type]` 文字内容与源流程一致。
- [ ] `@[wait]`、`@[move_route]`、`@[transport_event]` 顺序正确。
- [ ] `@[map_transport]` 的 `map` 不带 `res://` 和 `.tscn`。
- [ ] `@[enter_section]` 正确。
- [ ] 开关、变量、自开关命名正确。

## 动画与音频

- [ ] 所有动画来自 `D:\文字游戏`。
- [ ] 没有自创或临时替代动画。
- [ ] `.tscn` 动画路径正确。
- [ ] 序列帧 `path/frame_count/time_sec/target` 正确。
- [ ] 动画触发条件已记录。
- [ ] 音效路径正确。
- [ ] BGM 和环境音设置正确。
- [ ] 缺资源时已写入风险。

## 文字规则

- [ ] `sentence_rules` 格式正确。
- [ ] 句子规则能触发。
- [ ] 句子规则失效时能恢复状态。
- [ ] `split_list` 格式正确。
- [ ] `merge_list` 格式正确。
- [ ] 拆字、合字、推字、删字操作能完成。

## 运行检查

- [ ] Godot 打开场景无缺失资源报错。
- [ ] 从项目主入口能进入该地图。
- [ ] 关键流程至少手动跑一遍。
- [ ] 控制台无新增 Parser Error。
- [ ] 控制台无缺脚本、缺贴图、缺音效错误。

## 交接检查

- [ ] 已完成项写清楚。
- [ ] 未完成项写清楚。
- [ ] 风险写清楚。
- [ ] 下一位接手者知道从哪里继续。
