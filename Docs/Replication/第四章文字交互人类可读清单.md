# 第四章文字交互人类可读清单

> 本清单面向人工核对和复刻优先级判断；机器可读原始数据见 `第四章文字交互复刻数据.json`。

## 读法

- 坐标格式：`grid` 是全局格坐标，`page/local` 按 `32 x 18` 分页换算。
- `interactions` 里的 `press/touch/auto/loop` 来自事件触发方式，`push/delete/split` 是文字操作能力。
- `big_text` 需按行列摆放，`＿` 表示空格。
- 运行时生成文字不是地图初始字，要挂到打字/剧情系统。
